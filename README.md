# ElGamal Cipher (VCL/C++ Builder) — README

Этот проект представляет собой реализацию **криптосистемы Эль-Гамаля** на языке **C++ (C++ Builder / VCL)**. Программа позволяет:

- генерировать параметры системы (p, g, случайный k, секретный X и открытый Y);
- шифровать строку текста методом Эль-Гамаля;
- дешифровать ранее зашифрованное сообщение;
- визуально просматривать пары значений (a, b) и результат расшифровки.

---

## Используемый алгоритм

Программа реализует **классическую асимметричную криптосистему Эль‑Гамаля**, основанную на задаче дискретного логарифмирования в поле по модулю p.

### Параметры:

- **p** — большое простое число  
- **g** — первообразный корень по модулю p  
- **X** — секретный ключ  
- **Y = g^X mod p** — открытый ключ  
- **k** — случайный эпизодический ключ для каждого сеанса шифрования

### Шифрование символов

Каждый символ переводится в число:  
`m = charToValue(ch)`

Пара (a, b):

```
a = g^k mod p
b = m * (Y^k mod p) mod p
```

### Дешифрование:

```
M = b * a^(p−1−X) mod p
```

##  Возможности программы

- Выбор и ввод параметров p и g  
- Генерация ключей k, X, Y  
- Шифрование текста (построчно)  
- Просмотр результата в виде пар `(a, b)`  
- Дешифровка с восстановлением текста  
- Проверка корректности параметров  
- Поддержка кириллицы (буквы ‘а’–‘я’)

##  Как использовать

### 1. Введите значения:

- **p** — большое простое  
- **g** — первообразный корень  

### 2. Нажмите **Generate Keys** (Button3)

Будут получены:

- k — эпизодический ключ  
- X — секретный ключ  
- Y — открытый ключ  

### 3. Введите текст для шифрования (Memo1)

### 4. Нажмите **Encrypt** (Button1)

Появится список пар `(a, b)` в Memo2.

### 5. Нажмите **Decrypt** (Button2)

Расшифрованный текст появится в Memo3.

---

##  Ограничения

- Алфавит ограничен русскими буквами (а–я).
- p должно быть простым числом, g — первообразным корнем.
- Метод предназначен для учебных целей, **не рекомендуется для реальной криптографии**.
